# Test data for the schemaorg-arc-builder project

This directory contains test data files used for testing the system.

## Test Data Structure

- `sample_metadata.json` - Sample Schema.org JSON-LD metadata
- `complex_metadata.json` - Complex metadata with multiple entities
- `minimal_metadata.json` - Minimal metadata for basic tests

## Usage

Test data can be used in unit and integration tests to validate functionality
without needing external services like GitHub or GitLab.
